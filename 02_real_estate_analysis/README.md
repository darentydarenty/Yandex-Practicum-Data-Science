# Исследование объявлений о продаже квартир

## Описание проекта

На базе данных архива объявлений о продаже квартир в Санкт-Петербурге и соседних городах за несколько лет, нужно научиться определять рыночную стоимость объектов недвижимости. Первая задача — установить параметры. Это позволит построить автоматизированную систему: она отследит аномалии и мошенническую деятельность.
По каждой квартире на продажу доступны два вида данных. Первые вписаны пользователем, вторые — получены автоматически на основе картографических данных. Например, расстояние до центра, аэропорта, ближайшего парка и водоёма.

## Используемые инструменты

`matplotlib` `seaborn` `pandas` `numpy`
